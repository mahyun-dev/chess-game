# Chess AI Game

♟️ HTML + JavaScript로 만든 완전한 체스 게임  
🤖 4단계 난이도 AI (초급 ~ 전문가)  
💡 초보자 모드: 이동 가능한 위치 표시  
✨ 모던 그라디언트 UI 디자인  
🎯 힌트 시스템: 화살표로 최선의 수 표시  

## 실행 방법
1. `index.html` 파일을 브라우저에서 열기
2. 혹은 [여기를 클릭해서 실행하기](https://mahyun-dev.github.io/chess-game/index.html)

## 조작법
- **마우스 클릭**: 말 선택 및 이동
- **초급 난이도**: 말을 선택하면 이동 가능한 위치가 자동으로 표시됨
- **힌트 버튼**: 현재 최선의 수를 화살표로 표시 (5초간)
- **무르기 버튼**: 한 수 되돌리기
- **새 게임 버튼**: 게임 초기화

## 주요 기능
### ♟️ 완전한 체스 규칙
- 모든 기물의 정확한 이동 규칙
- 캐슬링 (Castling)
- 앙파상 (En Passant)
- 폰 프로모션 (Pawn Promotion)
- 체크/체크메이트/스테일메이트 감지

### 🤖 AI 난이도
1. **초급**: 랜덤 이동 + 이동 가능 위치 표시
2. **중급**: 2단계 깊이 탐색
3. **고급**: 3단계 깊이 탐색
4. **전문가**: 4단계 깊이 탐색

### 💡 초보자 지원
- 말 선택 시 이동 가능한 칸에 **초록 점** 표시
- 체크 상태일 때 **빨간 테두리** 경고
- 힌트 기능으로 최선의 수 안내

### 🎨 UI 특징
- 모던 그라디언트 배경
- 부드러운 애니메이션 효과
- 반응형 디자인 (모바일 지원)
- 직관적인 버튼 배치

## 파일 구조
```
├── index.html          # 메인 HTML 구조
├── styles.css          # 스타일시트
├── game.js             # 게임 초기화 및 렌더링
├── moves.js            # 각 기물의 이동 규칙
├── validation.js       # 체크/체크메이트 검증
├── actions.js          # 이동 실행 및 특수 동작
└── ai.js               # AI 로직 (Minimax 알고리즘)
```

## 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3**: 그라디언트, 애니메이션, Flexbox
- **Vanilla JavaScript**: 순수 자바스크립트로 구현
- **알고리즘**: Minimax with Alpha-Beta Pruning

## AI 알고리즘
- **Minimax 알고리즘**: 상대방의 최선의 수를 고려한 결정
- **Alpha-Beta Pruning**: 탐색 효율성 향상
- **평가 함수**: 
  - 기물 가치 평가 (폰=10, 나이트/비숍=30, 룩=50, 퀸=90, 킹=900)
  - 위치 보너스 (중앙 배치, 폰 전진 등)
